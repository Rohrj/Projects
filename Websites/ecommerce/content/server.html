Server Setup

The new domain name will point to a hosted server running, Anti-spam tools for staff emails, and weekly automatic backup using Updraft Pro. Content will be managed using Wordpress with Wordfence, Jetpack, and Akismet defensives plugged in.

Two administrators will have login credentials for the hosting, domain, and website accounts.

Success and maintenance of the website will be monitored using JetPack, WordFence, Google Search Console, Google Analytics, and Google Tag Manager.
